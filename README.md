# 德州扑克 (Texas Hold'em Poker)

一个用 Godot 4.4 制作的暗蓝色主题德州扑克游戏。

## 如何运行

1. 下载并安装 Godot 4.4: https://godotengine.org/
2. 克隆这个仓库或下载 ZIP
3. 在 Godot 中导入项目
4. 点击运行 (F5)

## 游戏规则

- 每人发两张底牌
- 三轮下注：翻牌前 → 翻牌 → 转牌 → 河牌
- 用最好的 5 张牌组合比大小

## 牌型大小

1. 皇家同花顺
2. 同花顺
3. 四条
4. 葫芦
5. 同花
6. 顺子
7. 三条
8. 两对
9. 一对
10. 高牌

## 操作

- **弃牌**: 放弃本轮
- **过牌**: 不下注
- **跟注**: 跟上下注
- **加注**: 增加下注

## 设计特点

- 暗蓝色主题界面
- 金色奖池显示
- 霓虹风格的按钮配色
